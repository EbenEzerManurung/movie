> A modern, performant movie streaming web app inspired by Netflix. Built with React, TypeScript, and Vite. Features full PWA support, offline caching, and advanced performance optimizations.


---

## ✨ Key Features

- 🎥 **Trending Movies** – Browse the latest popular movies from TMDB API  
- 🔍 **Real‑time Search** – Instant search with client‑side filtering  
- 📱 **Fully Responsive** – Optimized for mobile, tablet, and desktop  
- ⚡ **Blazing Fast** – Performance scores up to 49+ (Lighthouse) with ongoing optimizations  
- 📦 **PWA Ready** – Install as native app, works offline, caches API responses  
- 🎨 **Netflix‑like UI** – Smooth animations, skeleton loading, dark mode  
- ♿ **Accessible** – ARIA labels, keyboard navigation, semantic HTML  
- 🛠 **Developer Friendly** – TypeScript, lazy loading, code splitting

## 🚀 Tech Stack

| Category       | Technology                                                                 |
|----------------|----------------------------------------------------------------------------|

## 📦 Installation & Setup

### Prerequisites
- React.js (19.1.0)
- Node.js (v25 or later)
- npm or yarn
- TMDB API Key (free)

### Steps

Clone the repository
   ```bash
   git clone https://github.com/EbenEzerManurung/movie.git
   cd movie

   Navigate to the project directory:

```bash
cd movie
```

Restore dependencies:

```bash
npm install
```

Build the application:
```bash
npm run build
```

Run the application
```bash
npm run dev
```
